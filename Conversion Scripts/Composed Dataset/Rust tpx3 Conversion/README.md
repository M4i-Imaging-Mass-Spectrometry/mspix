# TPX3 File to MSPix file conversion

The associated "mspix.rs" file is part of a larger project that is not yet published or available in an open source format. This mspix.rs file is intended to demonstrate the conversion process of a fast mass microscopy (FMM) FMM-produced .tpx3 file format to the mspix file format. More information on reading FMM-produced .tpx3 files is provided in the Fast Mass Microscopy example code here:

https://github.com/M4i-Imaging-Mass-Spectrometry/fast-mass-microscopy-example
